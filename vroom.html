ABOUT

Vroom is a full-stack car rental platform built with Node.js and modern web technologies. The project features a robust backend system for managing vehicle inventory, user authentication, and booking management. The frontend delivers a seamless user experience with real-time availability updates, interactive booking calendar, and responsive design that works across all devices. The platform includes features like user reviews, vehicle filtering, and secure payment processing.
